## Asset Entry Format

Each asset uses exactly these 5 fields:

### Asset: [[Asset Name]]

- **Type:** [[Character sprite / Environment tile / UI element / Effect / Icon / Other]]
- **Dimensions:** [[Width × Height, e.g., 64×64, 1920×1080]]
- **Description:** [[Detailed narrative description of what the FINAL polished asset will depict in the shipped game. Describe the scene, character, or element as if briefing a concept artist: composition, key visual elements, lighting/mood, color harmonies, texture details, animation characteristics, and how it integrates into the broader visual style. Be specific enough that an artist could paint it or a 3D modeler could build it from this description alone.]]
- **References:** [[Links to reference images, existing games with similar assets, or concept art]]
- **Technical notes:** [[Sprite sheet layout, atlas requirements, file format, naming conventions, any engine-specific constraints]]
