Phase 4 draft: pillars and boundaries.

Proposed Pillars

Consequence-Rich Turns
Definition: Every turn must involve meaningful tradeoffs with irreversible tactical consequences.
Design test: If choosing between “safe deterministic move” vs “riskier line with strategic upside,” prefer the option that creates meaningful risk-reward evaluation.

Spectacle Through Strategy
Definition: Screen-filling projectile spectacle must emerge from planning, not reflex.
Design test: If choosing between adding reaction checks vs adding pre-turn planning depth, choose planning depth.

Build Identity Over Raw Stats
Definition: Distinct pattern synergies should define runs more than flat numeric scaling.
Design test: If choosing between +damage inflation vs a new interaction rule, choose interaction rule.

Always Multiple Valid Lines
Definition: The board state should usually present at least 2-3 competitively viable tactical plans.
Design test: If a feature funnels players into one dominant move sequence, redesign until alternatives are viable.

Proposed Anti-Pillars

We will NOT require twitch execution because it compromises Spectacle Through Strategy.
We will NOT rely on pure stat bloat progression because it compromises Build Identity Over Raw Stats.
We will NOT design “solved” encounters with one correct path because it compromises Always Multiple Valid Lines.
We will NOT overload the player with opaque rule exceptions because it compromises Consequence-Rich Turns.
